# Create Swap
A reusable script to create a 4GB swap. 

## How to use it:
1. Clone the repo `git clone https://github.com/darljed/script-create-swap.git` and `cd` into the folder.
2. Make the script executable executable: `chmod +x create-swap-ubuntu.sh` 
3. Run the script: `./create-swap-ubuntu.sh`

**NOTE:** *This script has only been tested in Ubuntu linux instance. Not yet tested in other linux distros*
